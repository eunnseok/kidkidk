>
> **아이가 직접 투자 항목에 대한 예측을 하여**
>
> **본인이 모은 도토리로 투자를 진행 할 수 있습니다.**

> **투자는 매일아침 9시에 결과가 나와**
>
> **투자 결과로 정산된 금액은 아이의 투자 주머니로 정산이 됩니다.**
